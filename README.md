# Portfolio Website

A modern, responsive portfolio website built with **React** and **SCSS**, showcasing a hero section, about section, and contact form. Fully responsive and optimized for desktop, tablet, and mobile devices.

## Features

- **Hero Section:** Profile image, name, subtitle, and CTA button with hover animation.  
- **About Section:** 2x2 grid of cards with skills and hover effects.  
- **Contact Section:** Responsive form with validation and success message.  
- **Dark Mode Support:** Compatible with light and dark themes.  
- **Responsive Design:** Optimized for desktop, tablet, and mobile devices.  
- **Modern Styling:** Hover effects, shadows, and smooth transitions.  

---

## Technologies

- **Frontend:** React  
- **Styling:** SCSS / CSS variables  
- **Bundler:** Vite  
- **JavaScript:** ES6+
